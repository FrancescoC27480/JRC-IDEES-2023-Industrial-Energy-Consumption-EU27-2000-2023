# JRC-IDEES-2023 Industrial Energy Consumption EU27 2000–2023

This project analyses industrial energy consumption across EU27 countries 
using the JRC-IDEES-2023 dataset.

## Data

The raw data is not included in this repository due to file size.

Download the JRC-IDEES-2023 dataset from the official JRC source:
👉 https://data.jrc.ec.europa.eu/dataset/1f0b480c-6d21-4d95-897d-20c7ca33df6f

Once downloaded, place the folder named `JRC-IDEES-2023` in the same folder 
as the notebook before running any code.
